# 자바스크립트에서 읽기 전용 제어자는 없다.

다른 언어에서는 `readonly` 접근제어자를 통해 읽기전용 속성을 선언할 수 있습니다.

하지만 자바스크립트에서는 불가능합니다.

`const`로 선언한 변수는 상수로 취급되어 값의 수정 및 번경이 불가능해집니다.

하지만 객체 속성을 const 로 선언하는것은 불가능합니다.

# 프로퍼티 디스크립터 사용

```
const a = {b:7}

Object.defineProperty(a,'b',{
    value:8,
    writable: false,
    enumerable: false,
    configurable: false
});
```
