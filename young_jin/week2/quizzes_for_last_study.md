### 3. 깂의 평가 순서

A, B 빈칸을 채우시오.

- A.

`+,-,/,%,>=,&&,??...`

` `방향으로 연산자가 평가됩니다.

- B.

`++,--, =, ++x, typeof...`

` ` 방향으로 연산자가 평가됩니다.

C~E: 값이 평가되는 순서대로 연산자의 종류를 쓰시오.
만약에 오류가 있는 코드라면 적절하게 괄호를 기입하여 오류를 해결하시오

예제
const x = a ?? b \*\* 2 || C

```
const x = (a ?? b ** 2) || c

연산순서
1. ** 지수 연산자
2. ?? 병합 연산자
3. || 논리 연산자
4. = 대입 연산지

```

- C.

```
const x = a ?? b || c;
```

- D.

```
const y = a && b ?? c;
```

- E.

```

```
